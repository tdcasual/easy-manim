## 核心特性

### 宫崎骏/吉卜力风格
- 柔和的天空渐变背景
- 手绘风格的装饰元素
- 自然、温暖的色彩
- 有机的圆角和形状

### 沉浸式创作
- 全屏无干扰界面
- 视频作为视觉中心
- 对话式输入（类似聊天）
- 生成中画笔动画反馈

### 折叠管理功能
- 任务管理 → 历史抽屉
- 视频库 → 历史抽屉
- 记忆/画像/评测 → 保留路由，通过链接访问

### 动画效果
- 云朵缓慢飘动（多层视差）
- 草丛微微摇摆
- 花瓣偶尔飘落
- 星星夜间闪烁
- 页面进入动画序列
