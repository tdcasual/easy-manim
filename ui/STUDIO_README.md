# 🎨 easy-manim Studio V3 - 宫崎骏风格创作界面

## 概述

全新的沉浸式 AI 动画创作界面，采用宫崎骏/吉卜力风格设计，将任务管理折叠到幕后，让创作成为主角。

## 设计理念

### 核心转变
- **从**: 任务管理系统 → **到**: AI 创作对话伙伴
- **从**: 信息过载 → **到**: 沉浸创作
- **从**: 冰冷科技感 → **到**: 温暖自然感

### 风格特征
- 🌤️ 柔和的天空渐变背景
- ☁️ 飘动的云朵动画
- ⭐ 夜间闪烁的星星
- 🌱 底部摇曳的草丛
- 🌸 偶尔飘落的花瓣
- 🖌️ 手绘风格的装饰元素

## 功能特性

### 1. 主题切换
- ☀️ **白天模式**: 温暖的晨光色调，蓝天白云
- 🌙 **夜间模式**: 梦幻的星空，月光和闪烁的星星

### 2. 氛围动画
- 云朵以不同速度缓慢飘动
- 草丛微微摇摆
- 花瓣偶尔飘落
- 星星夜间闪烁
- 所有动画支持 `prefers-reduced-motion` 减少动画

### 3. 交互反馈
- 输入提示词后"种子"飞行动画
- 生成中显示"画笔绘制"动画
- 完成后新视频淡入

### 4. 折叠的功能
- **历史记录**: 左侧滑出抽屉，显示所有任务
- **原有页面**: `/tasks`, `/videos`, `/memory` 等仍保留，可通过导航访问

## 路由变化

| 路径 | 说明 |
|------|------|
| `/` | 新的 Studio 创作界面（默认） |
| `/studio` | Studio 界面别名 |
| `/tasks` | 原有任务管理页面（保留） |
| `/tasks/:taskId` | 任务详情（保留） |
| `/videos` | 视频库（保留） |
| `/memory` | 记忆管理（保留） |
| `/profile` | 画像配置（保留） |
| `/evals` | 评测（保留） |

## 技术实现

### CSS 变量系统
```css
/* 白天 */
--sky-top: #E8F4F8;
--sky-bottom: #FFF8F0;
--grass-mid: #A5D6A7;
--accent-primary: #7CB342;

/* 夜间 */
--sky-top: #1A237E;
--sky-bottom: #3F51B5;
--star: #FFFDE7;
--accent-primary: #9575CD;
```

### 动画性能
- 使用 `transform` 和 `opacity` 进行动画
- `will-change` 优化
- 支持 `prefers-reduced-motion`
- Canvas 粒子数量自适应设备性能
